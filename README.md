# 🚧 Citizen Feedback on Road Maintenance

A full-stack web application developed during the **Samsung Innovation Campus Hackathon** to simplify the process of reporting road-related issues and improve communication between citizens and local authorities.

---

## 📌 About the Project

Citizen Feedback on Road Maintenance enables users to report road damages such as potholes, cracks, waterlogging, and other infrastructure issues. Citizens can upload images, provide location details, and track the status of their complaints through an easy-to-use interface.

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/YazhiniC03/Citizen-Feedback-Road-Maintenance.git
```

2. Install dependencies

```bash
npm install
```

3. Start the server

```bash
node backend/server.js
```

4. Open

```
http://localhost:3000
```
